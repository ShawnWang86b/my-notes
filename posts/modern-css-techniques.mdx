# Modern CSS Techniques: Grid, Flexbox, and Beyond

Modern CSS provides powerful tools for creating responsive, maintainable layouts. Let's explore the most important techniques.

## CSS Grid: Two-Dimensional Layouts

CSS Grid excels at creating complex, two-dimensional layouts:

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  grid-gap: 2rem;
  padding: 2rem;
}

.grid-item {
  background: #f5f5f5;
  padding: 1rem;
  border-radius: 8px;
}

/* Named grid lines */
.layout {
  display: grid;
  grid-template-columns: 
    [sidebar-start] 250px 
    [sidebar-end main-start] 1fr 
    [main-end];
  grid-template-rows:
    [header-start] 60px
    [header-end content-start] 1fr
    [content-end footer-start] 40px
    [footer-end];
}
```

## Flexbox: One-Dimensional Layouts

Flexbox is perfect for one-dimensional layouts and component alignment:

```css
.flex-container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
  gap: 1rem;
}

.flex-item {
  flex: 1; /* Grow and shrink equally */
}

.flex-item:first-child {
  flex: 0 0 200px; /* Don't grow/shrink, fixed 200px */
}

/* Centering technique */
.center {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
}
```

## CSS Custom Properties (Variables)

Create maintainable, dynamic styles:

```css
:root {
  --primary-color: #3b82f6;
  --secondary-color: #64748b;
  --spacing-unit: 1rem;
  --border-radius: 8px;
  --transition: 0.2s ease-in-out;
}

.button {
  background: var(--primary-color);
  color: white;
  padding: calc(var(--spacing-unit) * 0.5) var(--spacing-unit);
  border-radius: var(--border-radius);
  transition: all var(--transition);
}

.button:hover {
  background: color-mix(in srgb, var(--primary-color) 80%, black);
}

/* Dark mode with custom properties */
[data-theme="dark"] {
  --primary-color: #60a5fa;
  --background: #0f172a;
  --text: #f1f5f9;
}
```

## Container Queries

Responsive design based on container size:

```css
.card-container {
  container-type: inline-size;
  container-name: card;
}

.card {
  padding: 1rem;
}

@container card (min-width: 400px) {
  .card {
    display: grid;
    grid-template-columns: 1fr 2fr;
    gap: 2rem;
    padding: 2rem;
  }
}
```

## Modern Selectors

Use advanced selectors for cleaner markup:

```css
/* :has() - parent selector */
.card:has(.urgent) {
  border-left: 4px solid red;
}

/* :where() - specificity control */
:where(h1, h2, h3) {
  margin-top: 0;
  line-height: 1.2;
}

/* :is() - selector grouping */
:is(article, section) :is(h1, h2) {
  color: var(--heading-color);
}

/* nth-child with expressions */
.grid-item:nth-child(4n + 1) {
  grid-column: span 2;
}
```

## CSS Logical Properties

Future-proof layouts for international applications:

```css
.element {
  /* Instead of margin-left/right */
  margin-inline: 1rem;
  
  /* Instead of margin-top/bottom */
  margin-block: 2rem;
  
  /* Instead of border-left */
  border-inline-start: 2px solid var(--primary-color);
  
  /* Instead of text-align: left */
  text-align: start;
}
```

## Modern Units

Use modern CSS units for better responsive design:

```css
.container {
  /* Viewport units */
  min-height: 100vh;
  width: 100vw;
  
  /* Container query units */
  padding: 2cqi; /* 2% of container inline size */
  
  /* Dynamic viewport units (mobile-friendly) */
  height: 100dvh; /* Dynamic viewport height */
}

.typography {
  /* Fluid typography */
  font-size: clamp(1rem, 2.5vw, 3rem);
  
  /* Relative units */
  line-height: 1.6;
  margin-bottom: 1.5em;
}
```

## Performance Optimizations

Optimize CSS for better performance:

```css
/* Use transform and opacity for animations */
.fade-in {
  opacity: 0;
  transform: translateY(20px);
  transition: opacity 0.3s ease, transform 0.3s ease;
}

.fade-in.visible {
  opacity: 1;
  transform: translateY(0);
}

/* Contain layout shifts */
.image-container {
  aspect-ratio: 16 / 9;
  overflow: hidden;
}

.image-container img {
  width: 100%;
  height: 100%;
  object-fit: cover;
}

/* Use will-change sparingly */
.animated-element {
  will-change: transform;
}

.animated-element.animation-complete {
  will-change: auto;
}
```

## Conclusion

Modern CSS provides powerful tools for creating responsive, maintainable layouts. By mastering Grid, Flexbox, custom properties, and modern features, you can build better user interfaces with cleaner, more efficient code.
